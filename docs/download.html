I recognize that I should probably just use targets for this part, but… I (Matt Ross) don’t know how to use targets yet! So, instead we are just going to have a code chunk here that just downloads all the data directly from figshare, and the chunk will just be set to eval = FALSE, so it won’t run unless you manually run it.
